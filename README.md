## 😄 The Story Behind This

> You know that one friend who holds the camera and keeps saying...
>
> *"Smile!"* 😊
> *"Come on, SMILE!"* 😁
> *"BRO JUST SMILE PLEASE"* 😬
> *"WHY AREN'T YOU SMILING"* 🙏
>
> **Yeah. That was me. Every. Single. Time.**

So I built **SmileBuddy** — a camera that does the begging FOR me. 🎉
No smile? It **roasts you**. Out loud. Until you smile.

You're welcome. 😈

## ✨ Key Features

| Feature | What it does |
|---|---|
| 🎥 **Live Camera** | Opens right in the browser — no app, no install |
| 🤖 **AI Smile Detection** | face-api.js scans your face every few milliseconds |
| 📊 **Smile Score Bar** | Real-time 0.0 → 1.0 progress bar showing your smile level |
| 🔥 **5 Roast Levels** | Escalates from gentle to absolutely savage |
| 🔊 **Voice Roasts** | Text-to-Speech reads every roast OUT LOUD |
| 📸 **Auto Photo Capture** | Smile detected = photo taken. No button needed |
| 🎉 **Celebration Burst** | Confetti explosion the moment you smile |
| ⚙️ **Admin Panel** | Customize messages, colors, timing — all saved to database |
| 🗄️ **Live Database** | All content stored in Supabase — change anything without touching code |
| 🔒 **100% Private** | Face data never leaves your device. Ever. |

## ⚙️ How It Works

```
╔══════════════════════════════════════════════════════════╗
║  STEP 1 — App Opens                                      ║
║  SmileBuddy loads                                        ║
║  ➜ Fetches all messages, colors & settings               ║
║    from Supabase Database                                ║
╚══════════════════════╦═══════════════════════════════════╝
                       ⬇
╔══════════════════════════════════════════════════════════╗
║  STEP 2 — User Clicks START                              ║
║  ➜ Browser asks: "Allow Camera?"                         ║
║  ➜ Camera opens → Live video starts on screen            ║
╚══════════════════════╦═══════════════════════════════════╝
                       ⬇
╔══════════════════════════════════════════════════════════╗
║  STEP 3 — AI Starts Scanning                             ║
║  ➜ face-api.js scans face every few milliseconds         ║
║  ➜ Returns a SMILE SCORE from 0.0 to 1.0                 ║
║    0.0 = no smile  |  1.0 = big smile                    ║
╚══════════════════════╦═══════════════════════════════════╝
                       ⬇
╔══════════════════════════════════════════════════════════╗
║  STEP 4 — No Smile? Timer Starts!                        ║
║  ➜ Roast Level increases every few seconds               ║
║                                                          ║
║  Level 0 --> Level 1 --> Level 2 --> Level 3 --> Level 4 ║
║  (Green)   (Yellow)   (Orange)     (Red)     (Magenta)   ║
║                                                          ║
║  ➜ Message shows on screen                               ║
║  ➜ Voice speaks it ALOUD via Text-to-Speech              ║
╚══════════════════════╦═══════════════════════════════════╝
                       ⬇
╔══════════════════════════════════════════════════════════╗
║  STEP 5 — Smile Detected!                                ║
║  ➜ Smile Score crosses the threshold                     ║
║  ➜ Photo captured AUTOMATICALLY via HTML Canvas          ║
║  ➜ Celebration particles EXPLODE on screen               ║
║  ➜ Smile message plays — You survived!                   ║
╚══════════════════════════════════════════════════════════╝
```

## 🛠️ Tech Stack

```
📦 SmileBuddy
├── 🎨 Frontend         →  React 18 + TypeScript + Vite
├── 💅 Styling          →  Tailwind CSS
├── 🤖 AI Detection     →  face-api.js (TensorFlow.js)
├── 🗄️ Database         →  Supabase (PostgreSQL)
├── 🔊 Voice            →  Web Speech API (Text-to-Speech)
├── 📷 Camera           →  WebRTC getUserMedia
├── 🖼️  Photo Capture   →  HTML Canvas
└── 🎉 Animations       →  CSS + Particles
```

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/yourusername/SmileBuddy.git

# Go inside
cd SmileBuddy

# Install dependencies
npm install

# Add your Supabase keys in .env
VITE_SUPABASE_URL=your_url_here
VITE_SUPABASE_ANON_KEY=your_key_here

# Run it
npm run dev
```

> **Note:** Open this near friends who refuse to smile. Watch the chaos unfold. 😄

## 🔒 Privacy First

> No images uploaded. No face stored. No server involved.
> Just you, your face, and the friendly roasting. 😊
